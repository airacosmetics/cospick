## A. 이 문서에 대하여

이 문서는 COSPICK 의 **단일 진실 공급원(Single Source of Truth)** 이다.
앱의 각 페이지·기능을 구현·수정할 때마다 이 PRD 를 참조 문서로 사용한다.

- **참조 방법**: 섹션 번호(예: §3.2) 또는 마크다운 앵커 링크로 특정 부분만 콕 집어 참조한다.

- **관련 문서**: 디자인 방향(컬러·타이포·레이아웃)은 이 PRD 가 아니라 프로젝트 루트의 `DESIGN.md` 에서 관리한다. 프로젝트 규칙은 `CLAUDE.md` 를 참조한다.

## 1. 프로젝트 개요
COSPICK(코스픽)은 브랜드별로 흩어진 화장품 신상·가격·성분 정보를 **크롤링으로 한곳에 모아 비교**하고, 여기에 **뷰티 뉴스 큐레이션**과 **셀렉트샵 결제**까지 붙인 화장품 커머스 + 미디어 SaaS 다.
소비자는 검색 한 번으로 여러 브랜드의 제품을 비교하고, 최신 뷰티 트렌드 뉴스를 함께 확인한 뒤 바로 장바구니에 담아 결제한다.
데스크톱·모바일 웹 모두 지원하며, 구독(Pro) 회원에게 무제한 검색·프리미엄 큐레이션을 제공한다.

## 2. 타겟 페르소나
- **화장품 소비자 "지수" (28세)** — 여러 쇼핑몰을 오가며 성분·최저가를 비교하느라 지침. 페인포인트: "같은 제품인데 어디가 싼지, 성분이 내 피부에 맞는지 한눈에 안 보인다."
- **뷰티 크리에이터/MD "현우" (34세)** — 신상·트렌드를 매일 리서치해야 하는데 정보가 브랜드 사이트·뉴스·SNS 에 흩어져 있음. 페인포인트: "신상 출시와 업계 뉴스를 한 화면에서 빠르게 훑고 싶다."

## 3. 주요 기능 5가지

### 3.1 화장품 검색·비교 (AI)
- 자연어 질문("여드름에 좋은 진정 세럼 추천해줘")을 이해해 관련 상품을 추천 (Gemini AI, 실패 시 키워드 검색으로 폴백).
- 결과는 카드 그리드 — 브랜드·제품명·정가/할인가·성분 태그 비교.
- **사용자 흐름**: 검색바 입력 → AI 추천 결과 카드 → 카드 클릭 시 상세(성분·가격 비교) → 장바구니 담기.

### 3.2 크롤링 기반 신상·랭킹
- 주요 화장품 소스(브랜드몰·오픈마켓·뷰티 랭킹)를 **서버리스 크롤러**로 수집해 신상/인기 랭킹을 자동 갱신.
- 결과는 캐시(향후 Supabase/KV)에 저장, 프론트는 캐시를 조회.
- **사용자 흐름**: 랜딩/쇼핑 진입 → "이번 주 신상·랭킹" 섹션 자동 표시.

### 3.3 뷰티 뉴스 큐레이션
- 화장품/뷰티 업계 뉴스를 크롤링·수집해 `/news` 에 카드형으로 큐레이션 (제목·요약·출처·썸네일·링크).
- 카테고리 필터(신제품·트렌드·성분·브랜드소식) + 정렬(최신순).
- **사용자 흐름**: `/news` 진입 → 카테고리 선택 → 뉴스 카드 → 원문 링크 새 탭.

### 3.4 장바구니·결제
- 장바구니 담기/수량조절/합계는 서버(`/api/cart`)에서 계산 (localStorage 로 상태 유지).
- 결제는 Polar.sh (KRW, Merchant of Record) — `/checkout` 에서 테스트 카드 4242 로 결제.
- **사용자 흐름**: 장바구니 → `/checkout` → 결제 → 주문 완료.

### 3.5 구독 SaaS · 검색 한도
- 등급별 AI 검색/크롤링 조회 한도: **비로그인 하루 1회 · 로그인 무료 하루 5회 · Pro 무제한**, 한국시간 자정 초기화.
- Pro 구독(월 9,900원 / 연 99,000원) 시 무제한 검색 + 프리미엄 뉴스 큐레이션 + 신상 알림.
- **사용자 흐름**: 한도 초과 → 잠금 화면 → `/pricing` → 결제 → 무제한 해제.

## 4. 페이지 구조
- **랜딩 (`/`, index.html)** — 히어로(검색바 + 인기 키워드 칩) · 신상/랭킹(§3.2) · 인기 상품 그리드 · 뉴스 미리보기(§3.3) · 요금제 미리보기 · CTA · 푸터. 상태: 로딩/빈 상태/에러.
- **쇼핑·검색 (`/shop`, `/search`)** — 좌 필터(카테고리·피부타입·가격) + 우 상품 카드 그리드 + AI 검색 결과 메시지. 상태: 검색중/결과없음/한도잠금.
- **뷰티 뉴스 (`/news`)** — 카테고리 탭 + 뉴스 카드 그리드. 상태: 로딩/빈 상태/에러.
- **요금제 (`/pricing`)** — 무료 vs Pro 2 플랜 카드(월/연 토글, Pro 강조).
- **로그인·회원가입 (`/login`, `/signup`)** — 이메일 단독(Supabase Auth). placeholder 안내.
- **마이페이지 (`/mypage`)** — 헤더(구독 상태 배지) · 구독 카드 · 검색기록 · 즐겨찾기(찜한 상품).
- **결제 (`/checkout`)** — 주문 요약 + Polar 결제. 성공 시 `/mypage` "Pro 구독 중".

## 5. 성공 지표
- **가입 전환**: 랜딩 방문 → 회원가입 전환율 ≥ 8%.
- **결제 전환**: 가입 → Pro 결제 전환율 ≥ 3% (첫 30일).
- **재방문/리텐션**: 가입 후 7일 재방문율 ≥ 30%, 뉴스 페이지 주간 재방문 ≥ 20%.
