# 디자인 시스템 — COSPICK (로즈 베이지 뷰티 테마)

화장품 셀렉트샵에 어울리는 따뜻한 로즈·베이지 톤. 파일만 갈아끼우면 사이트 전체 분위기가 한 번에 바뀐다.

## 컬러 팔레트
- Primary(포인트): `#c98a7a` (로즈)
- Primary Deep: `#a9695b`
- Background: `#fbf7f4` / Card: `#ffffff`
- Text: `#2b2420` (ink) / Muted: `#8a7f78`
- Line/Border: `#ece4de`

## 타이포그래피
- 폰트: Noto Sans KR (한국어 전용)
- 제목: 32px / 700 / -1% letter-spacing
- 본문: 16px / 400 / 1.6 line-height
- 캡션·뱃지: 13px / 500

## 간격
- 기본 단위: 8px
- 섹션 사이: 80px
- 카드 내부: 20px

## 둥근 모서리
- 카드: 16px / 버튼: 999px(pill) / 입력창: 12px

## 그림자
- soft: 0 1px 3px rgba(0,0,0,0.05)
- medium: 0 6px 20px rgba(169,105,91,0.10)

## 컴포넌트 톤
- 버튼: pill 형태, 포인트 배경, 호버 시 Primary Deep 로 어두워짐
- 카드: 흰 배경 + soft 그림자 + 얇은 라인, 호버 시 살짝 부양
- 뱃지: 로즈 톤 반투명 배경 + 딥 로즈 텍스트 (예: NEW, BEST, 할인율)
- 뉴스 카드: 썸네일 상단 + 출처/카테고리 태그 + 제목 2줄 클램프
